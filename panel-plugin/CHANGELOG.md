# Changelog

## 0.1.0

Initial release. Native Grafana panel that renders the iPhone-style
storage breakdown bars from `nsd_node_storage_bytes`,
`nsd_node_filesystem_size_bytes`, `nsd_node_filesystem_used_bytes`,
and an optional Longhorn join.

Built against `@grafana/data` 12.4.x. Compatible with Grafana 10.4
and later.
